Ephemeral Grafana instance deployed successfully!

To access Grafana locally:
  export POD_NAME=$(kubectl get pods --namespace {{ .Release.Namespace }} -l "app.kubernetes.io/name={{ include "ephemeral-grafana.name" . }},app.kubernetes.io/instance={{ .Release.Name }}" -o jsonpath="{.items[0].metadata.name}")
  echo "Visit http://127.0.0.1:3000 to use Grafana"
  kubectl --namespace {{ .Release.Namespace }} port-forward $POD_NAME 3000:3000

Note: In production, this instance will be accessible via the ApplicationSet-generated ingress.
